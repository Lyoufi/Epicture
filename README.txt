Documentation:
Version 1.0

Application allowing you to manage an Imgur account, search for the images you want to see by writing a keyword, view the photos you have posted on your profile and log out when you want.

On the first page there is a presentation of the application that allows to know what are the functionalities who are on the application.

On the second page it is possible to search for the images you want through the search bar, you need to type a keyword and the images that refer to that keyword will appear on the page.

The third page allows to consult his profile (profile photo, pseudo) and thus the images posted on it.

Finally it is possible to disconnect from the application thanks to a button present in the drop-down menu at the top right of the application
